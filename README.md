# twstock

最近有一些換股需求，所以就開始開發一些小工具用一用
目前還沒有什麼 roadmap 就是了，畢竟有用才有想法
感謝 FinMind 提供的 API，太棒了!

## 工具列表

### twstock_chart_v2.html - 台股報價查詢工具
這是一個基礎的台股報價查詢工具，主要功能包括：
- 查詢指定股票代碼的最新收盤價和基本資訊
- 顯示最近 30 天的收盤價折線圖（使用 Chart.js）
- 列出最近 10 個交易日的詳細報價資料（開盤、最高、最低、收盤、成交量）
- 使用 FinMind API 取得台股歷史價格資料
- 響應式設計，自動調整圖表大小

### twstock_convert_v9.html - 台股換股計算器
這是一個進階的換股試算工具，主要功能包括：
- 同時查詢兩檔股票（來源股票與目標股票）的最新收盤價
- 顯示兩檔股票最近 30 天的漲跌幅比較折線圖（以百分比呈現）
- 計算換股試算：輸入來源股數，自動計算可換得的目標股數（無條件捨去至整股）
- 顯示換股前後的估算市值比較
- 股數顯示格式化（自動轉換為「張」和「股」的表示方式）
- 包含完整的投資風險免責聲明
- 使用 FinMind API 取得即時價格資料

兩個工具都採用：
- 純前端實作（HTML + JavaScript）
- Axios 處理 API 請求
- Chart.js 繪製互動式圖表
- 現代化的 UI 設計與卡片式佈局
